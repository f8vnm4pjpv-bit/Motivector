\## 分類する欲求（ココロ成分）

| 表示名  | 内部名・背景       |

| ---- | ------------ |

| 安定欲求 | 安全・損失回避・不安低減 |

| 自律欲求 | 自己決定・独立      |

| 成長欲求 | 有能感・達成       |

| 探究欲求 | 好奇心・認知欲求     |

| 関係欲求 | 所属・親和        |

| 評価欲求 | 承認・地位        |

| 影響欲求 | 権力・主導・作用     |

| 秩序欲求 | 構造化・予測可能性    |

| 刺激欲求 | 新奇性・快楽       |

| 創造欲求 | 表現・生成        |

| 意味欲求 | 自己超越・物語化     |





\## 各ココロ成分の説明

安定欲求：

不安定さや損失を避け、安心できる状態を保ちたい欲求。



自律欲求：

自分の意思で選び、自分のペースで動きたい欲求。



成長欲求：

できることを増やし、能力や成果を高めたい欲求。



探究欲求：

未知のものを知り、仕組みを理解したい欲求。



関係欲求：

人とのつながりを保ち、受け入れられていたい欲求。



評価欲求：

自分の価値や努力を認められたい欲求。



影響欲求：

自分の働きかけによって、周囲や状況に変化を起こしたい欲求。



秩序欲求：

情報や環境を整理し、見通しのよい状態にしたい欲求。



刺激欲求：

新しさ、面白さ、変化、楽しさを求める欲求。



創造欲求：

考えや感覚を形にし、何かを生み出したい欲求。



意味欲求：

出来事や行動に、物語・美学・思想・存在理由を見出したい欲求。



\## 作問ルール

1問に入れる欲求は、同じ場面で自然に起こりうるものに限定する

選択肢は欲求の説明ではなく、具体的な行動にする

同一設問内の選択肢は、行動・理由・違和感のどれかに粒度を統一する

関係欲求は「他者が存在する場面」でしか主配点にしない

探究欲求は「疑問・情報・未知」が自然に出る場面でしか主配点にしない

意味欲求は日常行動に混ぜすぎず、選択・継続・人生観の問いで測る

\## 表示するscore



素点

raw\_score\[欲求]

= 回答で得た点数の合計



欲求の強弱を測るのに用いる。

normalized\_score\[欲求]

= raw\_score\[欲求] / max\_possible\_score\[欲求]



ココロの成分表を表示するもの。メインの結果

component\_ratio\[欲求]

= normalized\_score\[欲求] / sum(normalized\_score\[全欲求])
